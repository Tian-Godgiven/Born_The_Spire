# Born The Spire 蘇生尖塔

一款基于纯js(ts)的《杀戮尖塔》同人游戏，以定制化牌组和角色性能为主要特色。

**原作**：https://www.nmbxd.com/t/62510110

语言：h5 + sass + ts  
平台：windows  
类型：PC端应用程序

## 简介

墙内的脉搏声停止，尖塔已然荒废，野兽、植物、失落的造物与被遗弃者徘徊在残桓断壁间。杳无人烟的塔底之下，已死高塔扎根的深处，响起了混沌的心跳。启程去唤醒这座高塔吧！ 

## 内容   

1. 以《杀戮尖塔》为基底，搭配原作精妙绝伦的创意，这将会是一款永久免费的同人游戏！ 
2. 在随机生成的阶层间冒险，击败敌人并将其吞噬，夺取种类各异的器官，不断强化角色能力！你还可以使用资源来不断升级你喜爱的器官！
3. 战斗中随机抽取器官提供的卡牌，组建各具特色的牌库，巧妙地运用各种能力击败强敌！ 
4. 灵活地搭配器官与随机获取的遗物，经历随机的事件，打败随机的强敌，如滚雪球般变得愈发强大，最终拼尽全力篡夺整座尖塔！  
5. 没有局外成长，但是每一局的付出均会转化为积分，使用积分永久解锁初始携带的器官，以更符合你心意的开局启程！  
6. 开放的游戏接口与相对完善的文档，我们欢迎每一个热爱创作的人为这款游戏添砖加瓦！

## 特色

原作在《杀戮尖塔》的游玩逻辑上进行了精妙的创新，而我们需要为这些特色注入更适合电子游戏的特征

1.物质：角色的“生命值”（以下称物质）将同时作为一种单向的资源，具有如下特征

- 不具备上限，归0时游戏失败(通常情况)
- 有着更多用途：在同化(获取)器官时损耗，修复损伤器官，升级已有器官，在商店换取金钱
- 获得物质的手段则包括：战斗胜利后选择吞噬敌人而非夺取器官，在水池(篝火)处通过唯一选项获得，击败每层的最终boss奖励

2.器官：角色能力的重要来源，种类各异的器官能够独特地强化角色的性能、填充角色牌库。并且这些器官还能够不断升级，进一步强化你的角色。不同器 官的效果搭配，往往能够产生惊人的效果！器官的获取途径包括：

- 从手下败将持有的随机3个器官中选择其一进行同化
- 在黑市中消耗金币购买指定的器官
- 经历独特的事件获取独特的器官
- 遗物，亦或者是某些器官本身的效果

3.黑市交易：随机出现在阶层间的黑市是玩家提升角色能力的重要地点，然而金币作为交易的硬通货，在荒废的高塔中却难以获取。玩家需要选择售卖角色拥有的器官或一定比例的物质来换取金钱。这些金钱能够保留，并且可以在任意黑市里换取各类商品：

- 各类器官，其中也包含黑市独有的商品，可以通过消耗金币刷新
- 一些珍贵的遗物，往往只会在这里出现
- 强大药水等一次性道具，在高塔荒废后，这些药水变得无比罕见

4.游玩奖励：根据玩家在每一局中达成的目标，在该局结束时奖励积分，这些积分可以用于永久解锁玩家在整个游戏过程中遇见的各类器官。在开局前，玩家可以让角色同化解锁的器官，该操作会略高地消耗角色开局时的物质。并且会按稀有度限制特定器官的数量。

# 详细介绍

注意：以下内容可能会在开发过程中修改，如有出入请参见最新版本。

## 1.界面UI

- 战斗：将显示敌我双方各自拥有的器官，并可以查看对应效果。
- 事件：事件的介绍文本与数个选项，支持响应小游戏界面。
- 爬塔：使用事件界面，按规则生成3个选项作为下一层的内容。选择其一进入下一层。

## 2.对象实例

### 角色

- 已死高塔根部孕育的不可名状之物质。通过吞噬与同化不断增强，最终夺取核心，复活并成为新的高塔。
- 物质：构成角色的内容物，承受来自外界伤害的同时也是一种珍贵的资源
- 吞噬：将任何实体化作自身的养料，得到大量的物质
- 同化：将他者的器官归为己用，并能够不断强化这些器官，这些过程都需要消耗一定量的物质
- 牌堆：角色的行动，每回合抽牌并打出卡牌以触发响应的行动效果

### 器官

功能：被动地提升角色性能，向角色的牌堆中提供卡牌，主动使用以提供效果  

用途

- 同化：消耗一定物质，令角色拥有该器官。消耗物质的量与器官本身，稀有度，以及器官是否损坏相关
- 吞噬：在非战斗状态下可选，消耗指定的器官并根据同化的消耗，等级和稀有度提供一定量的物质
- 售卖：在黑市出售器官以获取金钱，售出器官相较于售出物质的收益更高
- 升级：在水池处消耗一定物质以提升器官等级，器官的效果会随着等级不断提高
- 修复：消耗一定物质，使得已损坏的器官恢复其质量。

属性

- 部位：占据相同部位的器官互斥，玩家在选择新器官时会自动吞噬旧器官以获得一定量物质
- 等级：同化得到的器官固定为1级，通过在水池处消耗物质为器官升级。不同器官拥有不同的等级上限
- 稀有度：能够一定程度上反应器官的强力程度
- 质量：器官本身的生命值，敌人的一部分攻击会作用于器官，使其损失质量，当器官的质量归0时，该器官损坏，其不再提供被动效果，并且其提供的卡牌无法被打出，也无法使用其主动效果。

获取

- 战斗胜利后，可以选择从敌人拥有的器官中抽取3个作为选项以进行同化
- 部分事件会提供器官奖励
- 部分遗物，甚至是特殊器官的效果也会提供器官奖励
- 在黑市中消耗金钱获得指定的器官

### 水池

一个短暂的休憩处

在层级中概率出现的特殊房间，每层的数量在2~5个，在Boss房前固定出现。
玩家可以在这里选择进行一个行为，部分效果会提供额外的选项。最基本的行为包括：

- 汲取：吸收水池中物质，获得大量物质补给
- 升级：消耗自身的物质，提升器官的等级，该行为可以重复进行
- 染血：获得红色印记，该行为全局只能进行1次

### 卡牌

卡牌是角色在战斗中的行动，需要消耗相应的能量才能打出产生效果。每回合开始时抽一定数量的卡牌，回合结束时弃掉剩余的所有手牌。

获取：

- 一部分器官会向牌堆中添加一定数量的卡牌，这些卡牌与器官本身绑定，失去器官同时也会失去对应的卡牌，器官损坏时，相应的卡牌无法被打出。
- 一部分事件的选项会向牌堆中添加卡牌
- 一部分效果本身会添加卡牌，但其中包括仅在当前战斗中添加等类型

删除：

- 一部分事件的选项可以从牌堆中移除卡牌，即使这张卡牌由器官提供
- 一部分效果能够移除卡牌，其中也包含仅在当前战斗中移除的类型

### 敌人

在房间中遭遇的敌人与角色一样拥有各种器官，其性能不会发生变化。

敌人的行动会按照指定的行为规则使用器官提供的卡牌，并根据卡牌的效果显示为其下回合的意图。击破这些器官可以使其无法进行特定意图，这种情况下其会使用简单的攻击与防御。
此外，强大的敌人可能在一个回合中进行复数的行动。

精英敌人：在本层中也属于实力强劲的敌人，拥有强力器官的同时也会在击败时提供遗物奖励。
Boss敌人：看守着本层的终点，实力无比强大，并且往往具备无上限的强化能力，击败时可以指定地选择其1个器官进行同化，并且会掉落Boss级遗物

### 战斗

战斗为回合制进行，以角色的回合为起始，一直持续到一方的物质归0。

角色：每回合获得能量，抽取卡牌，打出手牌，丢弃手牌，期间可以选择使用遗物或器官的主动能力，亦或者使用药水。
敌人：每回合会规律地使用指定的器官卡牌作为行动，也会在指定的条件下使用器官的主动能力。

战斗奖励：在战斗胜利时，角色可以获得如下奖励

- 吞噬物质：获得一定量的物质，随着层级推进不断增加，强大的精英敌人会掉落更多物质
- 同化器官：从敌人拥有的器官中抽取3个，选择其1进行同化，也可以选择吞噬器官获得额外的物质
- 遗物奖励：仅限击败精英敌人时可选，从遗物池中随机抽取一个不重复的遗物奖励
- Boss器官：从击败的Boss敌人身上，指定地选择1个器官进行同化，与同化器官选项互斥
- Boss遗物：从3个随机抽取的Boss级遗物中选择1个

### 药水

一种能在任意时刻使用的道具，有着各式各样的效果。

由于高塔日渐荒凉，难以再在敌人身上获取药水了。最主要的获取渠道为在黑市购买，一部分器官/遗物/事件的效果也能够产生的药水。
不过，角色能够最大程度地吸收并发挥药水的效果，这导致原本效果平平的药水能够在战斗中发挥强大的效果，在非战斗情况下使用的药水也会为角色提供持续多次战斗的增益

### 遗物

被遗落在荒凉高塔中，但仍然具备力量的奇物。

能够在获取后的游戏全局中产生各种效果的强力道具，以被动效果为主，少数遗物能够主动使用。
遗物不会丢失，但部分遗物可能会在特定情况下失去效果。不会重复获得相同的遗物。

遗物具备稀有度和类型两种标识，类型决定其可能出现的场合，稀有度决定其出现的概率。

获取：

- 在黑市中使用金钱购买，其中也会出现独特的黑市遗物
- 击败精英敌人后抽取选择
- 击败Boss敌人后选择
- 一部分效果能够提供遗物
- 一部分事件的选项提供特殊的遗物，也存在抽取获取遗物的事件
- 在每一层固定出现的宝箱房中抽取选择

### 黑市与金钱

进行着可疑交易的窝点

为层级中概率出现的特殊房间，每层的数量为1~4个，在后半层出现的概率更大。
在黑市中，必须使用金钱来进行交易，金钱的获取方式包括：

- 在黑市中出售物质，随着售出次数增加不断贬值
- 在黑市中出售器官，根据器官的等级和稀有度决定售价
- 通过部分事件/遗物/器官的效果获取

金钱能够买到许多独特的物品或服务：

- 购买指定的器官，在黑市中购买同化器官不需要消耗物质，此外部分器官的等级可能会更高，价格由器官的稀有度和等级决定，一部分器官只能在黑市中购得
- 刷新黑市中出售的器官
- 购买仅在商店中随机出现的遗物，每次会出现3个，价格由遗物稀有度决定
- 删去角色牌堆中指定的卡牌，每个黑市仅允许删除1张，并且价格会愈发昂贵

### 事件

为层级中概率出现的特殊房间，每层的数量为3~7个。其中发生的事件会从当前层的事件池中抽取，不会重复。

事件具备不同的选项，对应产生不同的效果，可能具备随机性，这些效果都会显示在选项上，不会出现纯粹的惩罚性事件。事件能够提供的效果多种多样。

允许玩家通过SL的方式重置事件，但随机的结果并不会改变。

## 3.社区创作支持

### 接口

我们预计会开放以下内容的接口以供mod制作者使用，并会推出相应的使用文档

1. 角色，器官，卡牌，药水，事件，遗物，敌人，货币的全部接口
2. 黑市的部分接口，可以拓充黑市内可以能够的操作（商品的接口均设置在1当中）
3. 界面的部分接口，能够新增一些显示性的内容（比如鸡煲的球位，观者的变脸）和样式（比如某个效果的文本颜色/字体，效果内容的样式）。但UI的整体很难提供编辑的接口。

### 包管理系统

计划提供官方的mod包管理系统，用以辅助用户管理和使用mod

### mod制作工具

在框架的搭建结束，来到测试阶段时，会根据反响，考虑制作一个简单的mod制作辅助工具，提供一些预制化的选项，来帮助没有编程经验的朋友循序渐进地接触mod的制作
