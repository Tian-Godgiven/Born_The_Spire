# 角标系统

遗物和器官四个角上的小标签，用于展示冷却、计数、状态等动态信息。

## 基本概念

角标通过 `badges` 数组在遗物/器官数据中声明式配置。

每个角标有一个 `type`，决定默认位置、默认颜色和显示格式：

  indicator → 左上，蓝色，类型标识（⚡、文字等）
  state     → 右上，紫色，状态/模式（用 displayMap 映射显示）
  counter   → 左下，绿色，计数器（显示 "当前/最大"）
  cooldown  → 右下，灰色，冷却（0 时显示 ✓）

同一位置有多个角标时，在一行内依次排列。

## 自动角标

以下角标不需要配置，系统自动处理：

  ⚡ 主动能力标识（左上）：有 activeAbilities 时自动显示
  临时标识（右上，仅器官）：isTemporary 时自动显示

## 配置方式

在遗物/器官的数据定义中添加 `badges` 数组：

```typescript
{
    label: "节拍器",
    status: { "cooldown": 3 },
    badges: [
        { type: "cooldown", status: "cooldown" }
    ]
}
```

### BadgeConfig 字段

```typescript
interface BadgeConfig {
    type: "indicator" | "state" | "counter" | "cooldown"

    // 数据来源（选一个）
    status?: string           // 从自身 entity.status[key] 读值
    maxStatus?: string        // counter 的分母，从自身 status 读
    value?: string            // $ 引用表达式，如 "$owner.stateStack(power)"
    maxValue?: string | number // counter 的分母，支持引用或固定数字
    text?: string             // 固定文字

    // 条件显示
    showWhen?: Condition      // 条件表达式，如 "$scene == combat"

    // 显示覆盖
    position?: BadgePosition  // 覆盖默认位置
    style?: Record<string, string>  // 自定义样式
}
```

## 使用示例

### 冷却角标

冷却倒计时，0 时显示 ✓

```typescript
{
    status: { "cooldown": 3 },
    badges: [
        { type: "cooldown", status: "cooldown" }
    ]
}
```

### 计数角标

显示 "当前/最大" 格式

```typescript
{
    status: { "point": 0, "maxPoint": 10 },
    badges: [
        { type: "counter", status: "point", maxStatus: "maxPoint" }
    ]
}
```

用固定数字作为最大值：

```typescript
badges: [
    { type: "counter", status: "damage-taken", maxValue: 10 }
]
```

### 状态角标

自动读取 status 的 displayMap 映射显示文本

```typescript
{
    status: {
        "mode": {
            value: "even",
            displayMap: { "even": "偶", "odd": "奇" }
        }
    },
    badges: [
        { type: "state", status: "mode" }
    ]
}
```

### 固定文字角标

```typescript
badges: [
    { type: "indicator", text: "稀有" }
]
```

### 引用 owner 属性

使用 $ 引用语法读取拥有者的数据

```typescript
badges: [
    { type: "counter", value: "$owner.stateStack(power)" }
]
```

### 条件显示

使用条件表达式控制角标何时显示

```typescript
badges: [
    { type: "cooldown", status: "cooldown", showWhen: "$scene == combat" }
]
```

### 覆盖位置和样式

```typescript
badges: [
    {
        type: "counter",
        status: "point",
        maxStatus: "maxPoint",
        position: "top-right",
        style: { backgroundColor: "#ff0000" }
    }
]
```

### 多角标组合

```typescript
{
    label: "命运之轮",
    badges: [
        { type: "state", status: "mode" },
        { type: "counter", status: "drawCount", maxStatus: "drawThreshold" }
    ]
}
```

## 数据来源优先级

  text     → 固定文字，直接显示
  value    → $ 引用表达式，通过 ReferenceResolver 解析
  status   → 从自身 entity.status[key].value 读取

三者选一个。如果都没有，角标不显示。

## $ 引用语法

角标的 value 和 showWhen 使用统一的 $ 引用语法：

  $self.status(key)           → 自身属性
  $owner.status(key)          → 拥有者属性
  $owner.stateStack(power)    → 拥有者状态层数
  $scene                      → 当前场景类型（combat/pool/event）
  $battle.turn                → 当前战斗回合数

条件表达式支持比较操作符：

  "$owner.current(health) > 50"
  "$scene == combat"
  "$owner.current(health) <= 50%"
