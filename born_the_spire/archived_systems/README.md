# 归档系统

本目录包含已归档的系统实现，这些系统不再用于主项目，但保留用于未来的其他项目或作为参考实现。

## 已归档系统

### 1. 动态房间选择系统 (Dynamic Room Selection System)

**位置：** `dynamic_room_selection/`

**归档日期：** 2026-02-04

**归档原因：**
- 改用预生成地图系统（类似《杀戮尖塔》原版）
- 系统复杂度较高，概率平衡困难
- 玩家无法提前规划路线

**未来用途：**
- 用于其他项目（如计划中的"劲爆3选1无尽地牢战斗游戏"）
- 作为动态房间生成的参考实现

**文档：**
- [README.md](./dynamic_room_selection/README.md) - 系统详细说明
- [FILE_LIST.md](./dynamic_room_selection/FILE_LIST.md) - 文件清单
- [MIGRATION_GUIDE.md](./dynamic_room_selection/MIGRATION_GUIDE.md) - 迁移指南

**核心特性：**
- 基于规则的动态房间生成
- 多规则协调和冲突解决
- 房间池管理和耗尽策略
- 灵活的配置系统

## 使用归档系统

如果要在其他项目中使用归档系统：

1. 阅读系统的 README.md 了解架构和使用方法
2. 检查 FILE_LIST.md 了解文件结构
3. 参考 MIGRATION_GUIDE.md（如果有）了解集成方法
4. 复制相关文件到新项目
5. 根据新项目需求调整代码

## 注意事项

- 归档系统不再维护，可能与主项目最新版本不兼容
- 使用前请先测试并根据需要调整
- 建议作为参考实现，而不是直接复制使用
- 依赖项可能需要手动处理
