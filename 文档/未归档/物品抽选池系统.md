# 物品抽选池系统

物品通过 `pool` 字段标记其所属的抽选池，不同场景从不同的池中抽取物品。

## pool 字段

定义在 `ItemMap` 上，所有物品类型（卡牌、遗物、器官、药水）统一使用。

```typescript
// 在物品数据中定义
{
  key: "my_relic",
  label: "某遗物",
  pool: ["shop"],        // 商店限定
  // ...
}
```

pool 是字符串数组，一个物品可以属于多个池。

### 内置池名

  common：通用池，默认值。不写 pool 等同于 `["common"]`
  shop：商店池，仅在黑市出现
  boss：Boss池，仅在Boss战后出现
  elite：精英池
  event：事件池，仅通过特定事件获得

Mod 作者可以自定义池名，如 `["fishing"]`、`["sacrifice"]`。

### 默认行为

不写 pool 或 pool 为空数组，视为 `["common"]`。

```typescript
// 以下两种写法等价
{ pool: ["common"] }
{ }  // 不写 pool
```

建议始终显式写上 pool，提高可读性。

## drawItem 抽选函数

位置：`src/core/hooks/draw.ts`

从全局物品列表中按条件筛选并随机抽取一个物品。

```typescript
import { drawItem, drawItems } from "@/core/hooks/draw"
```

### drawItem 单次抽取

```typescript
drawItem(type, config?, customList?)
```

参数：
  type：物品类型 `"relic" | "card" | "organ" | "potion" | "custom"`
    使用 `"custom"` 时从 customList 中抽取
  config：抽选配置（可选）
    pool：限定池，字符串或数组，不指定时默认 `"common"`
    rarity：限定稀有度，字符串或数组
    tags：限定标签，字符串或数组
    exclude：排除的 key 列表（防重复）
    rarityWeights：稀有度权重，如 `{ common: 70, uncommon: 25, rare: 5 }`
    context：随机数上下文
  customList：自定义列表（仅 type 为 "custom" 时使用）

返回值：抽中的物品，无可选项时返回 null

### drawItems 多次抽取

```typescript
drawItems(type, count, config?, customList?)
```

自动排除已抽到的物品，不会重复。

### 使用示例

```typescript
// 抽一个商店遗物
drawItem("relic", { pool: "shop" })

// 抽一个罕见以上的通用遗物
drawItem("relic", { pool: "common", rarity: ["uncommon", "rare"] })

// 按稀有度权重抽通用遗物
drawItem("relic", { pool: "common", rarityWeights: { common: 70, uncommon: 25, rare: 5 } })

// 抽取时排除已有的
drawItem("relic", { pool: "shop", exclude: ["relic_001", "relic_002"] })

// 抽3个不重复的通用遗物
drawItems("relic", 3, { pool: "common" })

// 从自定义列表抽
drawItem("custom", { rarity: "rare" }, myMixedList)
```

## 黑市遗物栏位配置

位置：`src/static/config/blackStoreBalance.ts`

黑市的遗物不再按数量随机抽取，而是按栏位配置逐个抽取，每个栏位有独立的抽取规则。

### 栏位类型

```typescript
type RelicSlotConfig = DrawConfig | {
  chances: { weight: number, drawConfig: DrawConfig }[]
}
```

固定规则：直接传 DrawConfig
概率规则：通过 chances 按权重随机选择一条 DrawConfig

### 默认配置

```typescript
const defaultRelicSlots: RelicSlotConfig[] = [
  { pool: "shop" },                                    // 必定商店遗物
  { pool: "common", rarity: ["uncommon", "rare"] },     // 必定罕见以上
  {                                                     // 30%商店 / 70%通用
    chances: [
      { weight: 30, drawConfig: { pool: "shop" } },
      { weight: 70, drawConfig: { pool: "common" } },
    ]
  },
]
```

### 自定义黑市配置

```typescript
// Mod 作者可以在房间配置中覆盖
{
  type: "blackStore",
  relicSlots: [
    { pool: "shop" },
    { pool: "shop" },
    { pool: "common", rarity: ["rare"] },
    { pool: "common", rarityWeights: { common: 50, uncommon: 30, rare: 20 } },
  ]
}
```
