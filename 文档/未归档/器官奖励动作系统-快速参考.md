# 器官奖励动作系统 - 快速参考

## 什么是器官奖励动作？

战斗奖励中选择器官时，可以对器官执行不同的操作：
- 同化 🫀 - 获得器官
- 吞噬 ⚗️ - 获得物质
- 献祭 🔥 - 回复生命（需要遗物）

## 如何添加新动作？

### 1. 定义动作处理器

文件：`src/core/hooks/organRewardActions.ts`

```typescript
export const myAction: OrganRewardActionHandler = async (organ, player) => {
  // 你的逻辑
  return true
}
```

### 2. 注册动作

在 `initOrganRewardActions()` 中：

```typescript
organRewardActionRegistry.registerAction({
  key: "myAction",
  label: "我的动作",
  icon: "✨",
  description: "动作描述",
  priority: 70,
  enabled: (organ, player) => {
    return player.enabledOrganRewardActions.has("myAction")
  }
}, myAction)
```

### 3. 创建解锁遗物

文件：`src/static/list/item/relicList.ts`

```typescript
{
  label: "解锁遗物",
  key: "relic_unlock_myaction",
  interaction: {
    possess: {
      target: { key: "owner" },
      effects: [{
        key: "enableOrganRewardAction",
        params: { actionKey: "myAction" }
      }]
    },
    lose: {
      target: { key: "owner" },
      effects: [{
        key: "disableOrganRewardAction",
        params: { actionKey: "myAction" }
      }]
    }
  }
}
```

## 工作原理

```
获得遗物
  ↓
possess 交互触发
  ↓
enableOrganRewardAction effect
  ↓
player.enabledOrganRewardActions.add("myAction")
  ↓
动作出现在器官选择界面
```
