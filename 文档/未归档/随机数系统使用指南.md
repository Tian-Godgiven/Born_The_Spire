# 随机数系统使用指南

基于种子的确定性随机数生成系统，支持存档/读档和游戏回放。

## 核心类

### SeededRandom

位置：`src/core/utils/SeededRandom.ts`

基于 Mulberry32 算法的伪随机数生成器，相同种子产生相同序列。

```typescript
const rng = new SeededRandom(12345)
const value = rng.next()  // 0.6011037379503250
```

## 工具函数

位置：`src/core/hooks/random.ts`

提供高级封装，自动处理上下文和种子管理。

### 基础随机数

生成随机整数

```typescript
import { randomInt } from "@/core/hooks/random"

const damage = randomInt(5, 10, "cardDamage")  // [5, 10] 范围
```

生成随机浮点数

```typescript
import { randomFloat, randomFloatRange } from "@/core/hooks/random"

const chance = randomFloat("critChance")  // [0, 1)
const multiplier = randomFloatRange(0.8, 1.2, "damageVariance")  // [0.8, 1.2)
```

概率判断

```typescript
import { randomChance } from "@/core/hooks/random"

if (randomChance(0.3, "critChance")) {
    // 30% 概率触发
}
```

### 数组操作

随机选择单个元素

```typescript
import { randomChoice } from "@/core/hooks/random"

const card = randomChoice(availableCards, "selectReward")
```

随机选择多个元素（不重复）

```typescript
import { randomChoices } from "@/core/hooks/random"

const cards = randomChoices(allCards, 3, "discoverCard")
```

随机选择多个元素（可重复）

```typescript
import { randomChoicesWithReplacement } from "@/core/hooks/random"

const enemies = randomChoicesWithReplacement(enemyPool, 3, "spawnEnemies")
```

打乱数组

```typescript
import { randomShuffle } from "@/core/hooks/random"

const shuffledDeck = randomShuffle(deck, "shuffleDeck")
```

### 权重随机

根据权重选择单个元素

```typescript
import { randomWeightedChoice } from "@/core/hooks/random"

const rarity = randomWeightedChoice(
    ["common", "uncommon", "rare"],
    [70, 25, 5],
    "cardRarity"
)
```

根据权重选择多个元素（不重复）

```typescript
import { randomWeightedChoices } from "@/core/hooks/random"

const cards = randomWeightedChoices(
    allCards,
    cardWeights,
    3,
    "rewardCards"
)
```

### 直接使用随机数生成器

获取全局随机数生成器

```typescript
import { getGlobalRandom } from "@/core/hooks/random"

const rng = getGlobalRandom()
const value = rng.nextInt(1, 10)
```

获取带上下文的随机数生成器

```typescript
import { getContextRandom } from "@/core/hooks/random"

const rng = getContextRandom("discoverCard")
const randomCards = rng.shuffle(allCards).slice(0, 3)
```

## SeededRandom 方法

### 基础方法

next()
  生成 [0, 1) 范围的随机数

nextInt(min, max)
  生成 [min, max] 范围的整数

nextFloat()
  生成 [0, 1) 范围的浮点数

nextFloatRange(min, max)
  生成 [min, max) 范围的浮点数

chance(probability)
  根据概率返回 true/false

### 数组方法

choice(array)
  从数组中随机选择一个元素

choices(array, count)
  从数组中随机选择多个元素（不重复）

choicesWithReplacement(array, count)
  从数组中随机选择多个元素（可重复）

shuffle(array)
  打乱数组（返回新数组）

### 权重方法

weightedChoice(items, weights)
  根据权重随机选择一个元素

weightedChoices(items, weights, count)
  根据权重随机选择多个元素（不重复）

### 种子管理

getSeed()
  获取当前种子值（用于保存）

setSeed(seed)
  设置种子值（用于加载）

fork()
  创建新的随机数生成器，使用当前状态作为种子

derive(suffix)
  创建派生的随机数生成器（用于子系统）

## 上下文系统

上下文系统根据游戏状态自动生成唯一种子，确保相同状态下产生相同随机结果。

上下文组成

```
game:12345|floor:3|room:5|turn:2|ctx:discoverCard
```

  game: 游戏初始种子
  floor: 当前层数
  room: 当前房间索引
  turn: 当前回合数（战斗中）
  ctx: 自定义上下文

使用场景

卡牌效果中使用

```typescript
const rng = getContextRandom("discoverCard")
const randomCards = rng.shuffle(allCards).slice(0, 3)
```

为特定对象生成随机数

```typescript
const rng = getContextRandom(`card:${card.__id}`)
const damage = rng.nextInt(5, 10)
```

## 实际应用示例

### 事件效果中使用

```typescript
"gainRandomRelic": async (params?: { count?: number }) => {
    const count = params?.count || 1
    const { getLazyModule } = await import("@/core/utils/lazyLoader")
    const relicList = getLazyModule<any[]>('relicList')
    const { randomChoices } = await import("@/core/hooks/random")

    // 使用上下文随机选择
    const selected = randomChoices(relicList, count, "gainRandomRelic")

    for (const relicData of selected) {
        await doEvent({
            key: "gainRelic",
            source: nowPlayer,
            medium: nowPlayer,
            target: nowPlayer,
            effectUnits: [{
                key: "gainRelic",
                params: { relicKey: relicData.key }
            }]
        })
    }
}
```

### 卡牌效果中使用

```typescript
// 随机伤害卡牌
{
    key: "randomDamageCard",
    effect: async (event, effect) => {
        const { randomInt } = await import("@/core/hooks/random")
        const damage = randomInt(5, 10, `card:${effect.source.__id}`)

        await doEvent({
            key: "damage",
            source: event.source,
            target: event.target,
            effectUnits: [{ key: "damage", params: { value: damage } }]
        })
    }
}
```

### 战斗中使用

```typescript
// 随机生成敌人
const { randomWeightedChoice } = await import("@/core/hooks/random")

const enemyType = randomWeightedChoice(
    ["slime", "goblin", "orc"],
    [50, 30, 20],
    `spawnEnemy:${battleTurn}`
)
```

## 注意事项

确定性要求
  相同的初始种子 + 相同的调用顺序 = 完全相同的结果
  如果调用次数或顺序不同，结果会不同

上下文命名
  使用有意义的上下文名称，便于调试
  例如："discoverCard"、"spawnEnemy"、"cardDamage"

避免使用 Math.random()
  项目中所有随机操作都应使用 SeededRandom
  Math.random() 无法保存/加载，会破坏确定性

保存/加载
  保存时：`saveData.randomSeed = nowGameRun.rng.getSeed()`
  加载时：`nowGameRun.rng.setSeed(saveData.randomSeed)`
