# 进阶系统

进阶系统（Ascension System）用于提升游戏难度，通过修改敌人属性、奖励等来增加挑战性。

## 核心概念

进阶系统基于现有的触发器+效果架构实现，不引入新的系统。

工作原理：
  在游戏开始时，根据进阶等级应用一组触发器到玩家身上
  这些触发器监听 enemyCreation 事件
  当敌人创建时，触发器执行效果来修改敌人属性
  触发器在整局游戏中持续生效

## 类型定义

位置：`src/core/types/AscensionConfig.ts`

```typescript
export interface AscensionConfig {
    level: number
    triggers: TriggerMap[]
}

export interface SimpleAscensionModifiers {
    // 敌人属性修饰器
    enemyHealthMultiplier?: number      // 敌人生命倍率
    enemyDamageMultiplier?: number      // 敌人伤害倍率

    // 精英敌人修饰器
    eliteHealthMultiplier?: number      // 精英生命倍率
    eliteDamageMultiplier?: number      // 精英伤害倍率

    // Boss修饰器
    bossHealthMultiplier?: number       // Boss生命倍率
    bossDamageMultiplier?: number       // Boss伤害倍率

    // 奖励修饰器（未实现）
    cardRewardCount?: number            // 卡牌奖励数量修正（+/-）
    organRewardCount?: number           // 器官奖励数量修正（+/-）
    goldRewardMultiplier?: number       // 金币奖励倍率

    // 玩家修饰器（未实现）
    playerStartHealthMultiplier?: number  // 玩家初始生命倍率
    playerStartEnergyModifier?: number    // 玩家初始能量修正（+/-）
}
```

## 简化配置API

位置：`src/core/utils/ascensionHelper.ts`

```typescript
export function createAscensionModifier(config: SimpleAscensionModifiers): TriggerMap[]
```

将简化的修饰器配置转换为触发器配置。

示例：

```typescript
const triggers = createAscensionModifier({
    enemyHealthMultiplier: 1.5,  // 敌人生命 +50%
    enemyDamageMultiplier: 0.3   // 敌人伤害 +30%
})
```

生成的触发器结构：

```typescript
[{
    when: "after",
    how: "make",
    key: "enemyCreation",
    event: [{
        targetType: "eventTarget",
        key: "modifyStatus",
        effect: [{
            key: "multiplyStatusBase",
            params: {
                statusKey: "max-health",
                multiplier: 1.5
            }
        }, {
            key: "setCurrentToMax",
            params: {
                currentKey: "health",
                statusKey: "max-health"
            }
        }]
    }]
}]
```

## 条件效果

某些效果支持条件参数，用于仅对特定敌人生效。

支持的条件参数：
  onlyIfElite: 仅对精英生效
  onlyIfBoss: 仅对Boss生效

支持条件参数的效果：
  multiplyStatusBase - 乘法修改属性基础值
  setCurrentToMax - 设置当前值为最大值
  addStatusModifier - 添加属性修饰器

示例：

```typescript
{
    key: "multiplyStatusBase",
    params: {
        statusKey: "max-health",
        multiplier: 1.5,
        onlyIfElite: true  // 仅对精英生效
    }
}
```

实现位置：
  `src/core/effects/status/changeStatus.ts` - multiplyStatusBase, setCurrentToMax
  `src/core/effects/modifier/addModifier.ts` - addStatusModifier

## GameRun集成

位置：`src/core/objects/system/GameRun.ts`

```typescript
class GameRun {
    public towerFire: number = 0  // 当前进阶等级
    private _ascensionTriggerRemovers: (() => void)[] = []

    async applyAscensionTriggers(triggers: TriggerMap[]): Promise<void>
    clearAscensionTriggers(): void
}
```

applyAscensionTriggers 方法：
  清除之前的进阶触发器
  将新的触发器应用到玩家身上
  保存触发器移除函数，用于后续清理

clearAscensionTriggers 方法：
  移除所有进阶触发器
  清空移除函数列表

## 游戏启动集成

位置：`src/core/objects/game/run.ts`

```typescript
export async function startNewRun(seed?: string, ascensionLevel: number = 0)
```

启动流程：
  创建 GameRun 实例
  创建玩家实例
  如果 ascensionLevel > 0，应用进阶触发器
  设置 GameRun.towerFire 为进阶等级
  继续正常的游戏初始化流程

## 敌人创建事件

位置：`src/core/objects/target/Enemy.ts`

Enemy 构造函数在初始化完成后触发 enemyCreation 事件：

```typescript
constructor(map: EnemyMap) {
    // ... 初始化逻辑

    // 触发敌人创建事件
    doEvent({
        key: "enemyCreation",
        source: this,
        medium: this,
        target: this,
        effectUnits: []
    })
}
```

这个事件会触发玩家身上的进阶触发器，从而修改敌人属性。

## 使用示例

启动进阶5的游戏：

```typescript
import { startNewRun } from "@/core/objects/game/run"

await startNewRun(undefined, 5)
```

手动应用进阶触发器：

```typescript
import { nowGameRun } from "@/core/objects/game/run"
import { getAscensionConfig } from "@/static/list/system/ascensionList"

const config = getAscensionConfig(5)
if (config) {
    await nowGameRun.applyAscensionTriggers(config.triggers)
}
```

创建自定义进阶配置：

```typescript
import { createAscensionModifier } from "@/core/utils/ascensionHelper"

const customTriggers = createAscensionModifier({
    enemyHealthMultiplier: 2.0,  // 敌人生命翻倍
    eliteDamageMultiplier: 1.0   // 精英伤害翻倍
})

await nowGameRun.applyAscensionTriggers(customTriggers)
```

## 扩展指南

添加新的修饰器类型：

1. 在 SimpleAscensionModifiers 中添加新字段
2. 在 createAscensionModifier 中添加对应的触发器生成逻辑
3. 如果需要新的效果，在 effectMap 中注册
4. 如果需要新的事件，确保在适当的位置触发

添加新的进阶等级：

1. 在 ascensionList 中添加新的配置对象
2. 使用 createAscensionModifier 生成触发器
3. 测试新等级的难度平衡
