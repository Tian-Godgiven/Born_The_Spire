# 器官部位系统

## 概述

器官部位系统用于管理角色身上的器官位置和互斥关系。部位系统支持灵活的配置，既可以使用预定义的常用部位，也可以使用任意自定义字符串。

## 部位类型

### 常用部位（枚举值）

使用 `OrganPartEnum` 中定义的常用部位：

```typescript
import { OrganPartEnum } from "@/core/types/OrganTypes"

{
    label: "心脏",
    part: OrganPartEnum.Heart,  // 最多 1 个
    // ...
}

{
    label: "肺",
    part: OrganPartEnum.Lung,  // 最多 2 个
    // ...
}
```

常用部位列表：
  Heart (心脏) - maxCount: 1
  Lung (肺) - maxCount: 2
  Liver (肝脏) - maxCount: 1
  Stomach (胃) - maxCount: 1
  Brain (大脑) - maxCount: 1
  Eye (眼睛) - maxCount: 2
  Skin (皮肤) - maxCount: 3
  Bone (骨骼) - maxCount: 3
  Muscle (肌肉) - maxCount: 3
  Blood (血液) - maxCount: 1
  Nerve (神经) - maxCount: 2
  Gland (腺体) - maxCount: 3
  Core (核心) - maxCount: 1

### 独特部位（任意字符串）

可以使用任意字符串作为部位标识，用于修饰和渲染气氛：

```typescript
{
    label: "神秘的第三只眼",
    part: "mystical_third_eye",  // 独特部位
    // 未在 partConfigList 中注册，默认 maxCount = 1
    // ...
}

{
    label: "堕落的印记",
    part: "corrupted_mark",  // 另一个独特部位
    // ...
}
```

特点：
  未在 partConfigList 中注册的部位默认 maxCount = 1
  用于创建独一无二的器官
  适合特殊设计、剧情相关的器官
  显示名称默认使用原始字符串

### 无部位限制

不设置 `part` 字段（或设为 `undefined`），器官可以无限重复获取：

```typescript
{
    label: "寄生虫",
    part: undefined,  // 或不写 part 字段
    // 可以无限获取，不受数量限制
    // ...
}

{
    label: "临时强化",
    // 不写 part 字段
    // ...
}
```

特点：
  不受部位互斥限制
  可以拥有任意数量
  适合可重复获取的器官、临时器官等

## 部位互斥机制

当玩家获取器官时，系统会检查部位限制：

  如果器官有 part 字段
    获取该部位的最大数量（maxCount）
    检查当前已拥有的相同部位器官数量
    如果达到上限，弹出确认框询问是否吞噬旧器官

  如果器官没有 part 字段
    直接获取，不检查互斥

示例：

```typescript
// 玩家已拥有心脏器官
player.organs = [
    { label: "石芯", part: "heart", ... }
]

// 尝试获取新的心脏器官
acquireOrgan(newOrgan)  // newOrgan.part = "heart"

// 系统检测到心脏部位已满（maxCount = 1）
// 弹出确认框："部位 heart 已满，获取 新心脏 将会吞噬旧器官 石芯，是否继续？"
```

## 部位配置

部位配置存储在 `partConfigList` 中：

```typescript
// src/static/list/target/organPart.ts

export const partConfigList: PartConfig[] = [
    {
        key: OrganPartEnum.Heart,
        label: "心脏",
        maxCount: 1,
        description: "生命的核心，只能装备一个"
    },
    {
        key: OrganPartEnum.Lung,
        label: "肺",
        maxCount: 2,
        description: "呼吸器官，可以装备两个"
    },
    // ... 更多部位
]
```

配置项说明：
  key: 部位标识（字符串）
  label: 显示名称
  maxCount: 该部位最多可装备的器官数量
  description: 部位描述（可选）

## 工具函数

### getPartConfig(part: string): PartConfig | undefined

获取部位配置，如果未注册则返回 `undefined`。

```typescript
const config = getPartConfig("heart")
// { key: "heart", label: "心脏", maxCount: 1, description: "..." }

const unknownConfig = getPartConfig("unknown_part")
// undefined
```

### getPartLabel(part: string): string

获取部位显示名称，如果未注册则返回原始字符串。

```typescript
const label = getPartLabel("heart")  // "心脏"
const unknownLabel = getPartLabel("mystical_third_eye")  // "mystical_third_eye"
```

### getPartMaxCount(part: string): number

获取部位最大数量限制，如果未注册则返回默认值 1。

```typescript
const maxCount = getPartMaxCount("heart")  // 1
const unknownMaxCount = getPartMaxCount("mystical_third_eye")  // 1
```

### registerPart(config: PartConfig): void

注册新的部位配置（用于 Mod 扩展）。

```typescript
registerPart({
    key: "custom_part",
    label: "自定义部位",
    maxCount: 2,
    description: "Mod 添加的新部位"
})
```

## 使用场景

### 常用部位

适用于标准的器官设计，有明确的解剖学意义：

```typescript
{
    label: "石芯",
    rarity: OrganRarity.Uncommon,
    part: OrganPartEnum.Heart,  // 标准心脏部位
    // ...
}
```

### 独特部位

适用于特殊设计、剧情相关的器官：

```typescript
{
    label: "堕落的印记",
    rarity: OrganRarity.Rare,
    part: "corrupted_mark",  // 独特的诅咒印记
    // 这个部位只能有一个，用于渲染特殊的诅咒效果
    // ...
}

{
    label: "神圣光环",
    rarity: OrganRarity.Rare,
    part: "holy_aura",  // 独特的神圣部位
    // 用于渲染光环效果
    // ...
}
```

### 无部位限制

适用于可重复获取的器官、临时器官等：

```typescript
{
    label: "寄生虫",
    rarity: OrganRarity.Common,
    part: undefined,  // 可以有多个寄生虫
    // ...
}

{
    label: "临时强化",
    rarity: OrganRarity.Uncommon,
    isTemporary: true,
    temporaryRemoveOn: "battleEnd",
    // 不设置 part，战斗期间可以多次获取
    // ...
}
```

## Mod 扩展

Mod 可以通过 `registerPart` 添加新的部位配置：

```typescript
import { registerPart } from "@/static/list/target/organPart"

// 注册新部位
registerPart({
    key: "mechanical_implant",
    label: "机械植入",
    maxCount: 3,
    description: "机械改造的植入位置"
})

// 在器官中使用
{
    label: "机械手臂",
    part: "mechanical_implant",  // 使用 Mod 添加的部位
    // ...
}
```

## 注意事项

  部位系统是可选的
    不是所有器官都需要部位
    不设置 part 的器官可以无限重复

  部位用于互斥，不是用于分类
    部位的主要作用是限制数量
    如果不需要限制数量，不要设置 part

  独特部位的默认行为
    未注册的部位默认 maxCount = 1
    这确保了独特器官的唯一性

  显示名称的处理
    注册部位使用配置中的 label
    未注册部位使用原始字符串
    确保未注册部位也能正常显示

  部位字符串的命名
    建议使用有意义的英文字符串
    避免使用特殊字符和空格
    示例："mystical_third_eye"、"corrupted_mark"
