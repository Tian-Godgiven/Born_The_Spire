# 水池行动扩展系统 - 快速参考

## 什么是水池行动？

水池房间中可以执行的操作：
- 汲取 💧 - 获得物质（内置）
- 升级 ⬆️ - 升级器官（内置）
- 染血 🩸 - 获得印记（内置）
- 锻炼 💪 - 消耗物质增加生命（需要遗物）

## 如何添加新行动？

### 1. 定义行动处理器

文件：`src/core/hooks/poolActions.ts`

```typescript
export const myAction: PoolActionHandler = async (player, context) => {
  // 你的逻辑
  return true
}
```

### 2. 注册行动

在 `initPoolActions()` 中：

```typescript
poolActionRegistry.registerAction({
  key: "myAction",
  title: "我的行动",
  icon: "✨",
  description: "行动描述",
  priority: 60,
  repeatable: false,
  enabled: (player, context) => {
    return player.enabledPoolActions.has("myAction")
  }
}, myAction)
```

### 3. 创建解锁遗物

文件：`src/static/list/item/relicList.ts`

```typescript
{
  label: "解锁遗物",
  key: "relic_unlock_myaction",
  interaction: {
    possess: {
      target: { key: "owner" },
      effects: [{
        key: "enablePoolAction",
        params: { actionKey: "myAction" }
      }]
    },
    lose: {
      target: { key: "owner" },
      effects: [{
        key: "disablePoolAction",
        params: { actionKey: "myAction" }
      }]
    }
  }
}
```

## 工作原理

```
获得遗物
  ↓
possess 交互触发
  ↓
enablePoolAction effect
  ↓
player.enabledPoolActions.add("myAction")
  ↓
行动出现在水池选项中
```
