# Born The Spire - 项目简介

## 项目概述

Born The Spire 是一款基于 Slay the Spire 风格的桌面卡牌 RPG 游戏，采用现代 Web 技术栈构建。

## 核心玩法

- **卡牌战斗系统**: 玩家通过打出卡牌对敌人造成伤害、治疗自己、附加状态等
- **爬塔模式**: 玩家在塔中探索不同的房间（战斗、黑市、水池、事件等）
- **器官系统**: 角色拥有器官，器官可以提供特殊能力和触发器
- **物品收集**: 收集卡牌、药水、遗物、金钱等物品
- **状态管理**: 通过状态和当前值系统管理角色的属性和资源

## 技术特点

- **事件驱动**: 所有的游戏行为都通过事件和效果系统实现
- **事务管理**: 相关事件会被收集到同一个事务中顺序执行
- **触发器系统**: 灵活的触发器机制支持复杂的游戏逻辑
- **响应式UI**: 基于 Vue 3 的响应式系统实现流畅的用户界面
- **模块化设计**: 清晰的架构分层，便于扩展和维护
