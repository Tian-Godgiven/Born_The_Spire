# Bug 列表

## 未修复

1. **结束回合按钮遮挡卡牌文本**
   现象: 结束回合按钮会遮挡手牌卡牌的文本
   方案: 限制卡牌的空间，或 hover 时将卡牌移到上层显示

2. **尖啸导致敌人攻击意图始终显示0**
   现象: 敌人被尖啸（减力量）作用后，攻击意图显示的数值始终为0
   可能原因: 临时力量减少效果在意图计算中持续生效，没有正确还原

3. **非战斗状态右键遗物弹出介绍页面**
   现象: 非战斗状态下右键一个可使用的遗物会弹出介绍页面
   期望: 介绍页面应该是 hover 时弹出，右键应为使用操作

4. **黑市无法出售物质**
   现象: 黑市中无法出售物质
   期望: 优先出售物质，物质不足时再出售最大生命值

5. **宝箱事件未告知奖励内容**
   现象: 宝箱事件没有告知用户奖励到底有什么，意味不明


9. **竞技场事件战斗后遗物不可用**
   现象: 从竞技场事件进入战斗后无法使用袖珍手枪遗物，见好就收后没有显示奖励弹窗
   备注: 最终胜利时有奖励弹窗，但奖励很少
